bbplayer

bbplayer.js is a basic HTML5 Audio player script that lets you add a playlist of multiple source tracks to the audio element and create your own audio control buttons.

00:00 / --:-- 
<div class="bbplayer">
  <button class="bb-rewind">rew</button>
  <button class="bb-play">play</button>
  <button class="bb-forward">ff</button>
  <span class="bb-trackTime">--:--</span> / 
  <span class="bb-trackLength">--:--</span>
  <span class="bb-trackTitle"></span>
  <audio>
    <source src="media/Blue Browne.mp3" type="audio/mpeg"></source>
    <source src="media/Blue Browne.ogg" type="audio/ogg"></source>
    <source src="media/Georgia.mp3" type="audio/mpeg"></source>

    HTML5 Audio Not Available
  </audio>
</div>
